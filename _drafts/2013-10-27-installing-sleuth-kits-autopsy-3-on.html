--- layout: post title: Installing The Sleuth Kit's Autopsy 3 on Linux date: '2013-10-27T11:11:00.002+09:00' author: Joshua tags: modified_time: '2013-10-27T11:11:22.904+09:00' blogger_id: tag:blogger.com,1999:blog-2701259639305045003.post-4983271509535445901 blogger_orig_url: http://www.cybercrimetech.com/2013/10/installing-sleuth-kits-autopsy-3-on.html --- The Sleuth Kit (TSK) is a very nice kit for doing digital investigation tasks. Previously, TSK's tools were cross platform; however, with the newest version (3) of TSK's front-end - Autopsy - the developers decided to make it Windows only. Since most digital investigators around the world are using Windows, I can understand why. However, for investigators that prefer Linux, all is not lost.

In this example I will be using Ubuntu.
First, install WNE:
apt-get install wine
 Next, download the Autopsy 3 installer from here.

With WINE installed, you can simply double-click on the downloaded .msi file. In my case, autopsy-3.0.8-64bit.msi
Autopsy setup wizard: Next -> next -> Install
When it is finished copying files, click "Finish".

Open up the "Configure WINE" tool.
Under the applications tab, change the "Windows Version" to "Windows 7"
