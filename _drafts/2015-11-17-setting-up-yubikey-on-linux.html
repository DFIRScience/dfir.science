A yubikey (from Yubico) is a piece of hardware (USB/NFC) that supports different types of authentication methods - Yubico OTP, smart card (PIV), FIDO U2F, static passwords and others. If you don't have a yubikey, check out the video below to learn a bit more what it is.


I've seen the yubikey before, and finally decided to really see what it is about. I purchased the Yubikey Neo because I was interested in the Neo's ability to use NFC (can use with mobile devices without USB), and especially because it supports GPG, which I am extremely interested in getting more police to use. My yubikey is a version 3, and a week after purchase they released the Yubikey 4 (without NFC, but with stronger GPG keys). It seems like most processes should be the same.

When you get the key, it is configured with a yubikey one time password. As-is you can immediately configure it with any services that support security keys (google, github, dropbox, etc.). However, there is a lot more you can do if you want to configure the security key.

In the yubico downloads, there are a few applications available. It may not be obvious to everyone (it wasn't to me) what each does, and what you may need. This is very basically what each does as far as I understand it now:
  • Authenticator - generates a time-dependent code when it sees the yubikey
  • Neo Manager - change connection mode of Yubikey Neo
  • Personalization Tool - allows configuration of many yubikey features (OTP/Static Password)
  • PIV Tools - manage certificates (Yubikey Neo / 4)
For this post, we are focusing on the Personalization Tool installed on Linux (Ubuntu).
