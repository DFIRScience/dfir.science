Requires:
pygobject-2.0: http://ftp.gnome.org/pub/GNOME/sources/pygobject/
Note: the newest version of pygobject requires a version of gobject-introspection above what is available in macports. Because of this I went with version 2.10.1, which built without incident.

PyGTK: http://www.pygtk.org/downloads.html
For me pygtk installed from source with the standard ./configure make && make install

Download Mobius Forensic Toolkit sources
tar -xvf mobiusft-0.5.14.tar.gz
sudo python setup.py install
