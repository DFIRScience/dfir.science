Go to http://pyqwt.sourceforge.net/download.html, and download and install all packages in order.

  1. Python
  2. Numpy
  3. QT mac 4.5.2
  4. sip
  5. PyQT mac

If you get an error about disutils when trying to install numpy, download and run this setup script: http://peak.telecommunity.com/dist/ez_setup.py

Also get:
python-magic: https://github.com/ahupp/python-magic
afflib: macports
