--- layout: single title: "Facebook Capture the Flag Platform Now Available" date: '2016-05-13T13:17:00.002+09:00' tags: - Infosec - CTF - Education modified_time: '2016-05-13T13:24:34.011+09:00' thumbnail: https://4.bp.blogspot.com/-gBVUt3KtqSo/VzVVW5Qt45I/AAAAAAAABDs/tZaXYkgs2MARhu8SgJPeosZQTg2vE8iGQCLcB/s72-c/fbctf.jpg blogger_id: tag:blogger.com,1999:blog-2701259639305045003.single-2720276994316457702 blogger_orig_url: https://DFIR.Science/2016/05/facebook-capture-flag-platform-now.html --- Facebook's hacking education platform and capture the flag is now available. See their release single here. Their goal is to educate about different types of web attacks by giving access to CTF infrastructure and letting more groups run hacking competitions. From their github repository:

  • Organize a competition. This can be with as few as two participants, all the way up to several hundred. The participants can be physically present, active online, or a combination of the two.
  • Follow setup instructions below to spin up platform infrastructure.
  • Enter challenges into admin page
  • Have participants register as teams
    • If running a closed competition:
      • In the admin page, generate and export tokens to be shared with approved teams, then point participants towards the registration page
    • If running an open competition:
      • Point participants towards the registration page
  • Enjoy!
I'm playing with it now, but it looks like it will be an amazing resource for students.
